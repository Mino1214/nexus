텔레그램 — 시드 잔고 발견 알림
아주 간단하게 설정하세요
1. 텔레그램에서 @BotFather 검색 후 /newbot 입력
2. 받은 봇 토큰을 아래 칸에 붙여넣기
3. 알림 받을 개인채팅 또는 그룹방에 봇 초대
4. 개인채팅이면 /start, 그룹방이면 @봇아이디 test처럼 봇을 한 번 불러주기
5. Chat ID 자동 찾기 버튼 누르면 아래 칸에 자동 입력됩니다
