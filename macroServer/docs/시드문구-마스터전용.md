# 🔒 시드 문구 마스터 전용 설정 완료!

## ✅ 변경 사항

**시드 문구는 이제 마스터만 볼 수 있습니다!**

### Before (이전)
```
- 마스터: 모든 시드 문구 볼 수 있음
- 매니저: 자기 소속 유저의 시드 문구 볼 수 있음 ❌
```

### After (현재)
```
- 마스터: 모든 시드 문구 볼 수 있음 ✅
- 매니저: 시드 문구 섹션 자체가 안 보임 ✅
```

---

## 🔧 수정된 코드

### 1. server.js (백엔드)

#### Before
```javascript
// 매니저도 자기 유저의 시드 볼 수 있었음
app.get('/api/admin/seeds', requireAdmin, (req, res) => {
  if (req.admin.role === 'manager') {
    // 매니저는 자기 소속 유저만
    const myUserIds = new Set(...);
    list = seedStore.getAll(masked).filter(...);
  } else {
    // 마스터는 전체
    list = seedStore.getAll(masked);
  }
});
```

#### After
```javascript
// 마스터만 접근 가능
app.get('/api/admin/seeds', requireAdmin, requireMaster, (req, res) => {
  const list = seedStore.getAll(masked);
  res.json(list);
});
```

### 2. admin.html (프론트엔드)

#### 시드 섹션 숨기기
```javascript
function showDashboard() {
  const isMaster = me.role === 'master';
  // 시드 섹션은 마스터만 보임
  document.getElementById('sectionSeeds').classList.toggle('hidden', !isMaster);
  ...
}
```

#### 시드 로딩 제한
```javascript
async function loadAll() {
  if (me.role === 'master') {
    loadTelegram();
    await loadUsers();
    loadManagers();
    loadSeeds(); // 마스터만 시드 로드
  } else {
    loadUsers(); // 매니저는 유저만
  }
  loadSessions();
}
```
