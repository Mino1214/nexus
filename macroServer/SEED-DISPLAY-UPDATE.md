# ✅ 시드 문구 표시 개선

## 변경 사항

### 1. 테이블 컬럼 수정
**이전:**
```
| No | 사용자 | 문구 | 시각 |
```

**이후:**
```
| No | 시드번호 | 사용자 | 밸런스 | 시각 |
```

### 2. 시드번호 복사 기능 추가
- **표시**: `#1 📋` (클릭 가능)
- **동작**: 클릭 시 시드 문구를 클립보드에 복사
- **피드백**: 복사 완료 알림 표시

```javascript
function copySeed(no, phrase) {
  navigator.clipboard.writeText(phrase).then(() => {
    alert(`✅ 시드 #${no} 복사 완료!`);
  });
}
```

### 3. 밸런스 표시
- **네이티브 토큰**: 소수점 4자리
- **USDT**: 소수점 2자리
- **형식**: `0.0123 + 45.67 USDT`
- **밸런스 없을 때**: `0`

```javascript
const balance = s.balance || 0;
const usdtBalance = s.usdt_balance || 0;
const totalBalance = balance + usdtBalance;
const balanceDisplay = totalBalance > 0 
  ? `${balance.toFixed(4)} + ${usdtBalance.toFixed(2)} USDT` 
  : '0';
```

## 사용 방법

1. **시드 복사하기**
   - 시드번호 (예: `#1 📋`) 클릭
   - 자동으로 클립보드에 복사됨
   - 알림창에 복사 확인 메시지 표시

2. **밸런스 확인**
   - 네이티브 토큰 + USDT 합산 표시
   - 잔액이 0이면 단순히 `0` 표시

## 기술 세부사항

### 클립보드 API
- **최신 브라우저**: `navigator.clipboard.writeText()`
- **구형 브라우저 Fallback**: `document.execCommand('copy')`

### 보안 고려사항
- 시드 문구는 마스터 계정만 볼 수 있음
- 복사 시 전체 문구가 클립보드에 저장됨
- 화면에는 시드 번호만 표시되어 보안 강화

## 적용 방법

```bash
sudo ./quick-update.sh
```

## 테스트

1. 브라우저에서 `Ctrl + Shift + R`로 강력 새로고침
2. 마스터 계정으로 로그인
3. 수신 시드 섹션에서 시드번호 클릭
4. 클립보드 복사 확인
5. 밸런스 표시 확인
