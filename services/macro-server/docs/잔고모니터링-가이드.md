# 💰 잔고 모니터링 시스템

## 📋 개요

지정된 폴더를 주기적으로 스캔하여 잔고가 0 이상인 경우 텔레그램으로 즉시 알림을 보내는 시스템입니다.

---

## ✨ 주요 기능

- ✅ **자동 스캔**: 10초마다 폴더 자동 검사 (설정 가능)
- ✅ **텔레그램 알림**: 잔고 발견 시 즉시 텔레그램 전송
- ✅ **중복 방지**: 이미 알린 잔고는 재전송 안 함
- ✅ **다양한 형식 지원**: JSON, TXT, LOG 파일 자동 파싱
- ✅ **백그라운드 실행**: PM2로 24시간 무중단 운영
- ✅ **서버 재시작 시 자동 실행**: PM2 startup 설정됨

---

## 🔧 설정

### 환경 변수 (선택)

`.env` 파일 또는 환경 변수로 설정 가능:

```bash
# 모니터링할 폴더 (기본값: balances/)
WATCH_FOLDER=/path/to/your/balances

# 스캔 주기 (기본값: 10초마다)
# 크론 형식: 초 분 시 일 월 요일
CRON_SCHEDULE="*/10 * * * * *"
```

### balance-monitor.js 내부 설정

`balance-monitor.js` 파일 상단의 `CONFIG` 객체에서 직접 수정 가능:

```javascript
const CONFIG = {
  // 모니터링할 폴더
  WATCH_FOLDER: '/home/myno/바탕화면/myno/macroServer/balances',
  
  // 텔레그램 봇 토큰
  TELEGRAM_BOT_TOKEN: '8549976717:AAH5_jqcGCHlmZgSBi4nJNxmyVCKQI8HboQ',
  
  // 텔레그램 채팅 ID
  TELEGRAM_CHAT_ID: '7358393745',
  
  // 스캔 주기 (크론 형식)
  CRON_SCHEDULE: '*/10 * * * * *',  // 10초마다
  
  // 잔고 파일 패턴 (정규식)
  BALANCE_FILE_PATTERN: /balance.*\.(txt|json|log)$/i,
  
  // 최소 알림 잔고
  MIN_BALANCE: 0,
};
```

## 🚀 사용 방법

### 1. 서버 시작
```bash
cd /home/myno/바탕화면/myno/macroServer
pm2 start balance-monitor.js --name balance-monitor
pm2 save
```

### 2. 서버 상태 확인
```bash
pm2 status balance-monitor
```

### 3. 로그 확인
```bash
# 실시간 로그
pm2 logs balance-monitor

# 최근 50줄
pm2 logs balance-monitor --lines 50 --nostream
```

### 4. 서버 재시작
```bash
pm2 restart balance-monitor
```

### 5. 서버 중지
```bash
pm2 stop balance-monitor
```

### 6. 서버 삭제
```bash
pm2 delete balance-monitor
```

## 🔧 고급 설정

### 스캔 주기 변경

#### 5초마다
```javascript
CRON_SCHEDULE: "*/5 * * * * *"
```

#### 1분마다
```javascript
CRON_SCHEDULE: "0 * * * * *"
```

#### 10분마다
```javascript
CRON_SCHEDULE: "0 */10 * * * *"
```

#### 매시간 정각
```javascript
CRON_SCHEDULE: "0 0 * * * *"
```

### 크론 형식 설명
```
┌────────────── 초 (0-59)
│ ┌──────────── 분 (0-59)
│ │ ┌────────── 시 (0-23)
│ │ │ ┌──────── 일 (1-31)
│ │ │ │ ┌────── 월 (1-12)
│ │ │ │ │ ┌──── 요일 (0-7, 0과 7은 일요일)
│ │ │ │ │ │
* * * * * *
```

### 최소 잔고 설정

0.01 이상만 알림:
```javascript
MIN_BALANCE: 0.01
```

### 파일 패턴 변경

모든 JSON 파일:
```javascript
BALANCE_FILE_PATTERN: /\.json$/i
```

wallet으로 시작하는 파일만:
```javascript
BALANCE_FILE_PATTERN: /^wallet.*\.txt$/i
```

## 📝 실제 사용 예시

### 시나리오 1: 매크로 지갑 모니터링

```bash
# 1. 매크로가 잔고를 /home/myno/wallets/ 폴더에 저장
# 2. balance-monitor.js 설정 변경
WATCH_FOLDER: '/home/myno/wallets'

# 3. 서버 재시작
pm2 restart balance-monitor
```

### 시나리오 2: 특정 토큰만 모니터링

`balance-monitor.js`의 `parseBalance` 함수 수정:

```javascript
// ETH만 필터링
if (data.token === 'ETH' && balance > CONFIG.MIN_BALANCE) {
  results.push({ ... });
}
```

### 시나리오 3: 여러 폴더 모니터링

각 폴더마다 별도 인스턴스 실행:

```bash
# 폴더 1
WATCH_FOLDER=/folder1 pm2 start balance-monitor.js --name monitor-folder1

# 폴더 2
WATCH_FOLDER=/folder2 pm2 start balance-monitor.js --name monitor-folder2

pm2 save
```
