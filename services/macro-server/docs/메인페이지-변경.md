# 🏠 메인 페이지 변경 완료!

## ✅ 변경 사항

**이제 `mynolab.kr` 접속 시 바로 관리자 페이지가 나옵니다!**

### Before (이전)
```
mynolab.kr → 환영 메시지 페이지 → 관리자 로그인 버튼 클릭 → admin.html
```

### After (현재)
```
mynolab.kr → 바로 관리자 로그인 페이지 (admin.html)
```

---

## 📝 수정된 파일

### 1. `server.js`
```javascript
// 메인 페이지(/)를 관리자 페이지로 리다이렉트
app.get('/', (req, res) => {
  res.redirect('/admin.html');
});
```

### 2. `nginx.conf`
```nginx
# 메인 페이지(/)를 관리자 페이지로 리다이렉트
location = / {
    return 301 /admin.html;
}
```

### 3. `mynolab.kr.conf`
```nginx
# 메인 페이지(/)를 관리자 페이지로 리다이렉트
location = / {
    return 301 /admin.html;
}
```

## 💡 참고사항

### index.html은 어떻게 되나요?
- **삭제하지 않았습니다**
- 직접 `/index.html`로 접속하면 여전히 볼 수 있습니다
- 필요하다면 나중에 삭제하거나 다른 용도로 사용 가능

### 원래대로 되돌리려면?
```bash
# server.js에서 다음 코드 제거:
app.get('/', (req, res) => {
  res.redirect('/admin.html');
});

# nginx.conf에서 다음 부분 변경:
location = / {
    return 301 /admin.html;  # 이 부분 삭제
}

location / {
    try_files $uri $uri/ /index.html;  # 이렇게 변경
}
```
